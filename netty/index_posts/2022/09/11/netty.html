Netty Start

Andy Sun 11 Sep 2022

1,Hello Netty

Server

new ServerBootstrap() // 1, Launcher, package netty component , start server
    .group(new NioEventLoopGroup()) // 2, BossEventLoop(slector only interest in accept event,thread) , WorkerEventLoop(selector only interest in read event ,thread)  . NioEventLoopGroup contains BossEventLoop and WorkerEventLoop, deal with accept ,read event . 16, deal with read event ,receive bytebuf 
    .channel(NioServerSocketChannel.class) // 3, choose implement of ServerSocketChannel
    .childHandler(// 4, boss is in responsible for connection, worker(child) is responsible for read and write .  childHandler is used for worker
  new ChannelInitializer<NioSocketChannel>() {// 5, ChannelInitializer(channel stand for communication(read,write) channel,Initializer is responsible for adding other handler)
        protected void initChannel(NioSocketChannel ch) { //12 , call it when connection created
           // 6, add handler
            ch.pipeline().addLast(new StringDecoder());//7, bytebuf => String   17, parse bytebuf to hello
            ch.pipeline().addLast(new SimpleChannelInboundHandler<String>() { // 8, user-defined handler
                @Override // read event
                protected void channelRead0(ChannelHandlerContext ctx, String msg) { // msg is string from step 7 . 18 get hello
                    System.out.println(msg); 
                }
            });
        }
    })
    .bind(8080); // 6, bind port to listen

Client

new Bootstrap() // 7, Launcher for socket client 
    .group(new NioEventLoopGroup()) // 8, add event loop
    .channel(NioSocketChannel.class) // 9, add implement of socket channel 
    .handler(// 10, add handler
  new ChannelInitializer<Channel>() { 
        @Override //12, it would be called when connection created
        protected void initChannel(Channel ch) {
            ch.pipeline().addLast(new StringEncoder()); 
            // 15 , call method of StringEncoder ,encode string to bytebuf
        }
    })
    .connect("127.0.0.1", 8080) // 11, connect server
    .sync() // 13, block until connection created 
    .channel() // get socket channel (stands for connection obj)
    .writeAndFlush(new Date() + ": hello world!"); // 14, send data to server
