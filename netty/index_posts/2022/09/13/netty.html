Sync()

public static void main(String[] args) throws InterruptedException {
    ChannelFuture future = new Bootstrap()
            .group(new NioEventLoopGroup())
            .handler(new ChannelInitializer<NioSocketChannel>() {
                @Override
                protected void initChannel(NioSocketChannel ch) throws Exception {
                    ch.pipeline().addLast(new StringEncoder());
                }
            })
            .channel(NioSocketChannel.class)
            // connect is a asynchronous unlocking method ,
            // nio thread do connecting job
            .connect("localhost", 8080);

    future.sync();// block current thread,
    // wait synchronously until connected event happen in nio thread

    Channel channel = future.channel();
    channel.writeAndFlush("hello world");
    System.out.println(channel);
}

public static void main(String[] args) throws InterruptedException {

            .connect("localhost", 8080);

    //bind callable method to nio thread
    future.addListener(new ChannelFutureListener() {
        @Override // connect event happen in nio thread , call this method
        public void operationComplete(ChannelFuture channelFuture) throws Exception {
            Channel channel = channelFuture.channel();
            channel.writeAndFlush("hello world");
            System.out.println(channel);
        }
    });
}

public static void main(String[] args) throws InterruptedException {
        ChannelFuture future = new Bootstrap()
                .group(new NioEventLoopGroup())
                .handler(new ChannelInitializer<NioSocketChannel>() {
                    @Override
                    protected void initChannel(NioSocketChannel ch) throws Exception {
                        ch.pipeline().addLast(new LoggingHandler(LogLevel.DEBUG));
                        ch.pipeline().addLast(new StringEncoder());
                    }
                })
                .channel(NioSocketChannel.class)
                .connect("localhost", 8080);

        Channel channel = future.sync().channel();

        new Thread(()->{
            Scanner scanner = new Scanner(System.in);
            while (true){
                String line = scanner.nextLine();
                if("q".equals(line)){
                    channel.close();//same as connect ,
                    // close is a asynchronous unlocking method too 
                    log.info("deal with some work after close");
                    break;
                }
                channel.writeAndFlush(line);
            }
        },"input").start();
    }

Future & Promise

promise extends netty future extends jdk future

  • jdk future -> it can only wait for result synchronously,get result utill the task end

  • netty future -> it can both wait for result synchronously and asychronously , but still get result utill the task end

  • netty promise -> can be container for both thread .

Function/name jdk Future netty Future Promise
cancel cancel task - -
isCanceled   - -
isDone Judge task done or not ,cannot get to know success or fail - -
get get result synchronously blocking - -
getNow - get result, no block,get null if the task is not diminished，throw error if task failed -
await - wait until task done, does not throw error when failed -
sync - wait task end synchronously blocking，throw error if task failed -
isSuccess -   -
cause - get error message, no block -
addLinstener - add callback , get result asynchronously -
setSuccess - - Set success result
setFailure - - Set error result

Jdk future

public static void main(String[] args) throws InterruptedException, ExecutionException {
        ExecutorService service = Executors.newFixedThreadPool(2);

        Future<Integer> future = service.submit(new Callable<Integer>() {
            @Override
            public Integer call() throws Exception {
                log.debug("execute now");
                Thread.sleep(1000);
                return 50;
            }
        });
        log.debug("wait result");
        log.debug("result is {}",future.get());
    }

public static void main(String[] args) throws InterruptedException, ExecutionException {

        EventLoop eventLoop = new NioEventLoopGroup().next();

        DefaultPromise<Integer> promise = new DefaultPromise<>(eventLoop);

        new Thread(()->{
            log.debug("start run");
            try {
                Thread.sleep(1000);
            } catch (InterruptedException e) {
                e.printStackTrace();
                promise.setFailure(e);
            }
            promise.setSuccess(1);
